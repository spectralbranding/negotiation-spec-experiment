# Data & Code Availability — Negotiation Spec Experiment (completed)

The experiment is complete (3 core arms + 4 robustness components; 4,920 dyads across three
model families). The empirical note and all artifacts are openly available.

- **Empirical note + code (GitHub):** https://github.com/spectralbranding/negotiation-spec-experiment
- **Full transcripts + outcomes (HuggingFace dataset):** https://huggingface.co/datasets/spectralbranding/negotiation-spec-experiment — DOI 10.57967/hf/9090
- **Archival record (Zenodo) — concept DOI (always cite):** https://doi.org/10.5281/zenodo.20595996
  (this version v1.0.0: 10.5281/zenodo.20595997)

Licenses: code MIT; data CC BY 4.0. Canonical pre-run freeze = git commit 2ed95ffd
(Study 1) + the Study-2 preregistration. The graded-headroom variants and the dose-response
analysis are reproducible from the published scripts (`code/gen_headroom_variants.py`,
`code/compute_headroom.py`).

**Reconstruction disclosure.** Scenarios and harness authored independently, informed by public
materials; not a one-to-one reproduction of any base study (base study public ref arXiv:2503.06416).
